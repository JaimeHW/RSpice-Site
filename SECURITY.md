# Security Policy

Please do not disclose a suspected vulnerability in a public issue.

Use GitHub's private vulnerability reporting flow from this repository's
Security tab. Include the affected URL, reproduction steps, impact, and any
supporting request or console details with credentials and personal data
removed.

This repository contains only static website source. Vulnerabilities involving
the simulator, authentication, customer circuits, licensing, or cloud APIs
should still be reported privately here until the dedicated product security
intake is published.
